## Overview

torchDEsolver is a PyTorch-based library for solving differential equations using deep learning techniques. It provides a flexible framework for defining and training neural networks to approximate solutions to complex differential equations, enabling researchers and engineers to tackle challenging problems in fields like physics, engineering, and finance. With its modular design and easy-to-use API, torchDEsolver streamlines the process of developing and deploying custom solvers, making it an ideal tool for anyone looking to leverage the power of deep learning for differential equation solving.

## Core features

**Feature 1**: `torch` is used as the primary deep learning framework for building and training models.
**Feature 2**: The project utilizes `PyTorch`'s `nn.Module` to define custom loss functions for various models.
**Feature 3**: The codebase employs `finitediffs` from `tedeous.utils` for finite difference calculations.
**Feature 4**: The project uses `torch.nn.functional` for various functional operations, including `F.mseloss`.
**Feature 5**: The code is designed to work with various devices, utilizing `tedeous.device` for device management.

## Installation

Install torch_DE_solver using one of the following methods:

**Build from source:**

1. Clone the torch_DE_solver repository:
```sh
❯ git clone https://github.com/andreygetmanov/torch_DE_solver
```

2. Navigate to the project directory:

3. Install the project dependencies:
