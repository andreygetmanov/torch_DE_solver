## Overview

torchDEsolver is a PyTorch-based project that solves partial differential equations (PDEs) using neural networks. It provides a flexible framework for defining and training neural networks to approximate PDE solutions, with a focus on accuracy and efficiency. Key features include support for various loss functions, including mean squared error with derivatives, and compatibility with different devices, including GPUs. The project is ideal for researchers and engineers working with PDEs in fields such as physics, engineering, and computer science.

## Core features

1. **PyTorch Integration**: Utilizes `PyTorch` for building and training neural networks to solve partial differential equations.

2. **Modular Loss Functions**: Provides a base class for different loss functions, including mean squared error with and without derivatives.

3. **Multi-Derivative Support**: Supports calculation of loss functions with first, second, and third derivatives for improved accuracy.

4. **Device Flexibility**: Allows training on different devices, including GPUs, using `PyTorch`'s device management capabilities.

5. **Partial Differential Equation Solving**: Designed to solve partial differential equations using neural networks and various optimization algorithms.

## Installation

Install torch_DE_solver using one of the following methods:

**Build from source:**

1. Clone the torch_DE_solver repository:
```sh
❯ git clone https://github.com/andreygetmanov/torch_DE_solver
```

2. Navigate to the project directory:

3. Install the project dependencies:
