## Overview

torchDEsolver is a Python package that solves differential equations using the Differential Evolution algorithm, a global optimization technique. It provides a flexible and efficient way to find solutions to complex differential equations, making it a valuable tool for researchers and engineers in various fields. By leveraging the power of PyTorch, it enables fast and accurate computations, allowing users to focus on modeling and analysis rather than numerical implementation. With its user-friendly API and robust optimization capabilities, torchDEsolver is an ideal choice for solving a wide range of differential equations, from simple to complex systems. Its applications span across physics, engineering, biology, and finance, making it a versatile and powerful tool for anyone working with differential equations.

## Core features

1. **Deep Learning Framework**: `torch` is used as the primary deep learning framework for building and training models.

2. **Numerical Computation**: `numpy` is utilized for efficient numerical computations, enabling fast and accurate calculations.

3. **Scientific Computing**: `scipy` is employed for scientific computing tasks, providing functions for tasks such as optimization and signal processing.

4. **Data Analysis and Visualization**: `pandas`, `matplotlib`, and `seaborn` are used for data analysis and visualization, allowing for the exploration and presentation of results.

5. **Solution Evaluation**: The project includes a solution evaluation module, which assesses the accuracy and performance of solutions at given points.

## Installation

Install torch_DE_solver using one of the following methods:

**Build from source:**

1. Clone the torch_DE_solver repository:
```sh
❯ git clone https://github.com/andreygetmanov/torch_DE_solver
```

2. Navigate to the project directory:

3. Install the project dependencies:
