## Overview

<overview>
torchDEsolver is a software framework that helps researchers and practitioners solve complex differential equations, providing a comprehensive toolkit for approximating solutions to various physical problems. It offers a modular and flexible approach, enabling users to tackle a wide range of equations and systems, making it a valuable resource for fields like physics, engineering, and mathematics.
</overview>

## Core features

<corefeatures>

1. **Modular Design**: Separate modules for data processing, model definition, and optimization.

2. **Numerical Methods**: Implements various methods for solving PDEs and ODEs.

3. **Optimization Algorithms**: Includes particle swarm, Newton's method, and gradient descent.

4. **Data Processing Tools**: Provides data loading, preprocessing, and visualization capabilities.

5. **Callbacks and Evaluators**: Monitors optimization progress and evaluates solution quality.

</corefeatures>

## Installation

Install torch_DE_solver using one of the following methods:

**Build from source:**

1. Clone the torch_DE_solver repository:
```sh
❯ git clone https://github.com/andreygetmanov/torch_DE_solver
```

2. Navigate to the project directory:
```sh
❯ cd torch_DE_solver
```

3. Install the project dependencies:


**Using `pip`** &nbsp;
[<img align="center" src="https://img.shields.io/badge/Pip-3776AB.svg?style={badge_style}&logo=pypi&logoColor=white" />](https://pypi.org/project/pip/)

```sh
❯ pip install -r requirements.txt
```
